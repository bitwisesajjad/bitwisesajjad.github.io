Privacy Policy

Last updated: 29 April 2026

1. Scope

This privacy policy describes how Motify, a personal software project, handles data. Motify is operated privately by its owner and does not provide a service to end users. There are no user accounts and no public sign-up.

2. Data we collect

Motify does not collect personal data from members of the public. The software runs locally on the operator's machine and interacts with third-party APIs using credentials owned by the operator.

The only data Motify stores are: video files generated by the operator, scripts written by or for the operator, OAuth tokens authorising posting to the operator's own social media accounts, and logs of the operator's own activity within the application.

3. Data we do not collect

Motify does not collect, store, or process information about viewers, followers, or any third parties beyond what the relevant social media platforms expose through their public APIs (for example, aggregate view counts on the operator's own videos).

4. Third-party services

Motify sends data to third-party APIs in order to function. These include: the Anthropic API (for script generation), ElevenLabs (for voiceover), Pexels (for stock footage), and the official APIs of YouTube, TikTok, and Instagram (for publishing). Each of those services has its own privacy policy that governs how it handles the data sent to it.

5. Storage and security

OAuth tokens and API keys are stored locally on the operator's machine and are not shared. The repository excludes credential files via .gitignore.

6. Children

Motify is not directed at children and does not knowingly collect data from any individual.

7. Changes

This policy may be updated. The "Last updated" date at the top of this page reflects the most recent revision.

8. Contact

For privacy questions, contact the project operator via the GitHub repository at github.com/bitwisesajjad/motify.
